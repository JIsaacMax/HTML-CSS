Utilização de listas

Ordenadas

  1. Entregues por tag li
  2. Devem ser inseridas todas as vezes que um novo item é adicionado
  3. Se organiza conforme as tags propostas no código
  4. Numeração automática respeitando como foi descrito no código
  5. A ordem é relevante
  1. É possível selecionar o tipo de "numeração" que a lista terá, declarando dentro da tag ol o "type"
  2. Não é obrigatório o fechamento de tag em li, porém vai de acordo com seu próprio método de identação
  3. Já a tag ol ainda possuirá o fechamento
  4. Também é possível modificar a partir de que item a lista começa a contar. Ex: Ao invés do primeiro item receber 1 ele pode começar a contar com 4, usando a tag start logo após o type.

Dentro de um dl que é basicamente a tag que chama as definições
Por que disso?
Basicamente o HTML vai interpretar o que estiver escrito diferente quando você declara que ele é diferente. Por isso esse tipo de termo, por mais que visualmente igual, ele se torna diferente para o interpretador

Anotações de interesse!

Nomeclatura

O ol significa organized list, ou seja, lista organizada, já o ul é de unorganized list, lista desorganizada, assim fica mais facil de lembrar

Macetes

Modificar em massa os componentes é só clicar no meio da tag pra que ela fique pontilhada, segurar alt e ir clicando nas outras que vai querer mudar, depois de selecionar é só escrever que ele modifica tudo pra ti.
Até os fechamentos.
